10–15 Minute Playtest Goal

You do not need to finish the whole game. For v0.6, the most useful feedback is whether the new first-time flow feels less overwhelming and whether the accessibility settings help.

1

Create a character

Pick any origin, class, and keepsake. Tell us if character creation feels interesting or confusing.

2

Check Settings

Try larger text, bolder text, high contrast, or simplified ASCII if readability is an issue.

3

Talk to Mara

Visit People of Ashmere and speak with Mara Vell. Notice what unlocks afterward.

4

Explore The First Road

Try at least one Old Road event, then return to Ashmere and see what town options unlocked.

What We Need Feedback On

v0.6First 10 minutesAccessibilityMenu UnlocksMobileBugs
  • Did the starting Ashmere menu feel less overwhelming?
  • Did you understand what to do after entering Ashmere?
  • Did the locked menu options make sense?
  • Did the accessibility settings help with readability?
  • Did character creation feel like it mattered?
  • Was The First Road fun, too slow, too random, or confusing?
  • Did anything break or fail to respond?
  • Would you play another update?

Brutally honest feedback helps more than praise. If something is boring, confusing, inaccessible, or broken, say it.

Known Early-Build Notes

  • v0.6 is a refactor build and may be leaner than v0.5.1 in some systems.
  • The game now uses separate CSS and JavaScript files instead of one giant HTML file.
  • Art is mostly placeholder/ASCII style right now.
  • Balance is still changing quickly.
  • Mobile support exists, but real-device feedback is especially useful.
  • The game saves in your browser, and Save Vault can export/import portable saves.
