10–15 Minute Playtest Goal

You do not need to finish everything. The most useful feedback is whether the first session feels clear, playable on phone, and more like a real RPG than the older builds.

1

Create a traveler

Start a new save. Try origin, class, keepsake, and the new Traveler Interview. Tell us if the choices feel meaningful.

2

Watch the intro

Check whether the cinematic intro explains Ashmere, the road, and the recovered-save mood without dragging too long.

3

Reach Ashmere

Use the Start Here guide. Speak with Mara, then check whether the first town flow makes sense.

4

Travel and fight

Use Travel the Old Road, try a choice check, and test combat options besides Attack. Open inventory if you can.

What We Need Feedback On

v0.7.0MobileCharacter CreationOld Road ChecksCombatInventoryDefeatBugs
  • Did the title screen have too many buttons or feel okay on your phone?
  • Did the new character creation make your traveler feel more personal?
  • Did the intro help explain the world, or did it slow you down?
  • Did the Ashmere Start Here guide tell you what to do next?
  • Did Old Road choices feel better than random text-box rolls?
  • Did combat feel more interactive than just clicking Attack?
  • Did the improved inventory make sense?
  • If you died, did the defeat page and item/gold loss feel fair?
  • Did anything still show weird ASCII, clip, overlap, or break on mobile?
  • Would you keep playing after the first 10 minutes?

Brutally honest feedback helps more than praise. If something is boring, confusing, inaccessible, ugly, unfair, or broken, say it.

Known Early-Build Notes

  • This is still an early HTML5 browser RPG build.
  • Some v0.7.0 systems are additive layers over the restored v0.6.4 core for safety.
  • The risky universal theme overlay was disabled before release; the current build prioritizes stability.
  • Legacy ASCII art is being hidden/removed for the public build, but report any that still appears.
  • Balance is still changing quickly.
  • Mobile feedback is especially important for this weekend push.
  • The game saves in your browser, and Save Vault can export/import portable saves.
