Devlog 004 / Recovered Build v0.4.1

Mobile Polish

v0.4.1 is a small quality-of-life patch focused on making Legend feel better on phones and smaller screens.

This is not a big content update. It is a playability patch so the game can be smoother across desktop, tablet, and mobile.

What changed

  • Updated the viewport for safer mobile scaling.
  • Improved tap target sizes for buttons.
  • Added sticky mobile action controls.
  • Reduced mobile spacing and panel size.
  • Made ASCII panels scale better on narrow screens.
  • Made the top HUD more compact on phones.
  • Kept battle actions easier to reach on mobile.
  • Reduced excess scrolling where possible.

Why this matters

Legend is a browser RPG, so it should feel playable wherever someone opens it. This patch moves the project closer to that goal before bigger v0.5 systems are added.
