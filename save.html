How This Works

  • Browser save: the normal game still autosaves locally so it works instantly.
  • Portable save: this page turns that save into a file/code you control.
  • Import: paste or upload the save on another device/browser, then return to LEGEND and continue.
  • True cloud saves: will require a backend service, login system, or a platform API later.
