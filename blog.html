Devlog 008 / Recovered Build v0.6.4

Stability Refactor and Road Systems

v0.6.4 is the safety pass after the public build push. The main game file has been restored after an accidental stub overwrite, shared version labels now report v0.6.4, Save Vault was caught up, and the new refactor files were checked before being activated.

The goal is stability first: keep LEGEND playable while preparing modular systems for tutorial, Sella, NPC metadata, Old Road events, and future travel expansion.

Devlog 007 / Recovered Build v0.6

Refactor and Accessibility

v0.6 moved LEGEND out of one giant file, added progressive menu unlocks, and introduced accessibility settings for readability, contrast, motion, spacing, and ASCII display.
